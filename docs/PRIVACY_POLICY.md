# Privacy Policy

ClassMate AI is privacy-first and local-first.

## Data processed

The extension processes study content only after the student requests capture or imports a source. Study artifacts, notes, embeddings, sync metadata, provider settings, and cached retrieval data are stored locally unless a user enables a sync-backed workflow.

## Provider credentials

Provider credentials are stored in local extension storage and are not synced by default. Logs redact secret-like metadata.

## Network access

Network requests are made only for selected AI providers, optional backend services, and user-enabled sync or collaboration flows.

## Contact

Use the project repository issue tracker for privacy questions before release.
